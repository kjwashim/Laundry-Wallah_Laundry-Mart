# 🧺 Laundry Wallah / Laundry Mart

A simple and user-friendly web page for a laundry service business. This project demonstrates basic HTML structure including services, pricing, and a booking form.

## 📖 About the Project

This project is a **basic frontend web application** built using HTML.
It is designed to showcase how a laundry service website might look and function.

Users can:

* View available laundry services
* Check pricing details
* Submit booking information

---

## ✨ Features

* 🏠 Welcome section with introduction
* 🧼 List of laundry services
* 💰 Price table for services
* 📅 Booking form (Name, Email, Phone)
* 📩 Contact information

---

## 🛠️ Built With

* HTML5
* (Optional) CSS3

## 🚀 Getting Started

Follow these steps to run the project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/kjwashim/Laundry-Wallah_Laundry-Mart.git
   ```

2. Open the folder:

   ```bash
   cd Laundry-Wallah_Laundry-Mart
   ```

3. Run the project:

   * Open `index.html` in your browser
